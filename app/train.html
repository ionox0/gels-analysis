Training

1. Choose a JPG file with an image of Gel on which to train
2. Align lanes using rotation buttons, highlight first lane using the cursor
3. Hit 1 to label lane as Control Lane, 2 to label as Condition. Adjust crop region as necessary
4. Repeat steps 2 and 3 for all lanes
5. Select "Upload Training Lanes" to upload these labeled training examples
